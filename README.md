# Информационно-справочная система театра #

Веб-приложение для автоматизации управления театральным репертуаром. Система предоставляет инструменты для контроля постановок, анализа статистики посещаемости, управления пользователями и безопасного доступа к данным. Разработана в рамках курсовой работы по дисциплине «Современные технологии программирования» в Финансовом университете при Правительстве РФ.

## Функциональные возможности ##

**Аутентификация и авторизация**

1) Регистрация/вход с валидацией данных и шифрованием паролей (BCrypt).

2) Ролевая модель: USER (просмотр афиши, статистики) и ADMIN (управление постановками и пользователями).

3) Защита от CSRF-атак.

**Управление постановками**

1) CRUD-операции для театральных событий.

2) Фильтрация и сортировка по названию, жанру, дате, возрастному рейтингу.

3) Динамические запросы к БД через JPA Specifications.

**Административный модуль**

1) Управление пользователями (назначение ролей, удаление аккаунтов).

2) Защита от самоудаления и изменения собственной роли администратора.

**Статистика**

1) Визуализация распределения постановок по месяцам.

2) Анализ количества зарегистрированных пользователей.

**Пользовательский интерфейс**

1) Адаптивные шаблоны (Thymeleaf + Bootstrap 5).

2) Динамические компоненты: навигационная панель, фильтры, сортировка.

## Технологический стек ##
**Backend:**

*Java 21, Spring Boot 3.4 (Security, Data JPA, Cache), REST API, ETag, PostgreSQL*

**Frontend:**

*Thymeleaf, HTML5/CSS3, JavaScript, Bootstrap 5*

**Инструменты:**

*Maven, Spring Cache, JPA Specifications*

## Архитектура ##

Система построена по многоуровневой архитектуре:

1) Презентационный слой:

*Thymeleaf-шаблоны + REST API (ApiController).*

2) Бизнес-логика:

*Сервисы (PerformanceService, UserService, StatsService).*

3) Доступ к данным:

*Spring Data JPA, репозитории (PerformanceRepository, UserRepository).*

4) Безопасность:

*Spring Security, BCrypt, CSRF-токены.*
